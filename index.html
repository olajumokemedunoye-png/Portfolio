# HR Attrition Analysis

## Business Problem
Identify patterns driving employee attrition and understand the financial cost of turnover.

## Tools Used
| Tool | Purpose |
|------|---------|
| SQL (PostgreSQL) | Data extraction & transformation |
| Excel | Data cleaning & cost modelling |
| Power BI | Dashboard & visualisation |

## Dataset
- 847 employee records
- Fields: Department, Tenure, Salary, Overtime, Performance, Exit Status

## Key Insights
- 68% of exits occurred within the first 18 months (onboarding failure)
- Sales department had the highest attrition rate at 31%
- Employees working overtime showed significantly higher turnover risk
- Each mid-senior exit cost an estimated £18,500 in total replacement cost

## SQL Queries
See `hr_attrition.sql` for full query set including:
- Attrition rate by department
- Tenure band segmentation
- Overtime vs attrition correlation
- Cost-of-turnover calculation

## Dashboard
![HR Attrition Dashboard](dashboard.png)

## Impact
- Identified £312K in annualised turnover cost
- Flagged 3 high-risk cohorts for proactive retention
- Recommended 90-day check-in programme for early-tenure employees
